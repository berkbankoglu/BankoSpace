## ğŸ¯ Getting Started

### Sign Up / Sign In
- Create an account to sync data across devices
- Or use "Continue Without Login" for offline-only mode

### Your First Todo
1. Click "Add Todo" at the top
2. Type your task and press Enter
3. Click the checkbox to mark complete (with animation!)

### Create a Goal
1. Go to the Goals section (left sidebar)
2. Click "+ Add Goal"
3. Set your target and track progress

### Start a Pomodoro Timer
1. Find the Timer section
2. Set your work duration (default: 25 min)
3. Click "Start" and focus!
